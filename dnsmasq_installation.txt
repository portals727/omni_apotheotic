ðŸ”§ Installation Instructions
Method 1: Online Installation (Router has internet)
If your router can temporarily connect to the internet (e.g., via Ethernet or as a Wi-Fi client), use the apk command
bash
Update the package list
apk update

Install dnsmasq-full dependencies automatically
apk add dnsmasq-full
Method 2: Offline Manual Installation (No internet)
If the router has no internet access, you must manually download the packages on another computer and transfer them to the router.
Go to the base directory
https://downloads.openwrt.org/snapshots/packages/aarch64_cortex-a53/base/
Download dnsmasq-full-2.91-r2.apk
Crucially, you must also download all dependency packages listed above. 
Transfer packages to the router
Use SCP (e.g., scp *.apk root@192.168.8.1:/tmp/) or copy them to a USB drive and mount it on the router.
SCP
scp *.apk root@192.168.8.1:/tmp/
Install on the router
Navigate to the directory containing the .apk files
cd /tmp

Install all packages 
apk add --allow-untrusted *.apk
